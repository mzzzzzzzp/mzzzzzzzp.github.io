# 注解
## 自动装配
* 组件扫描
    - @Component：表示这个类需在应用程序中被创建
    - @ComponentScan：自动发现应用程序中创建的类 
- 自动装配
    - @Autowired：自动满足bean之间的依赖 
- 定义配置
    - @Configuration：表示当前类是一个配置类
