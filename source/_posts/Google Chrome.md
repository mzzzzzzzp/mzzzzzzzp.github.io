### 快捷键
| Ctrl + F4 | 关闭当前标签页 |
| :--: | :--: |
| Alt + F4 | 关闭所有窗口 |
| Ctrl + T | 打开新标签页 |
| Ctrl + Shift + T | 恢复标签页 |
| Ctrl + N | 打开新的窗口 |
| Ctrl + Shift + N | 开启无痕模式 |
| Ctrl + Tab | 多个标签页之间切换 |
| Alt + ←/→ | 后退或前进 |
| Ctrl + Shift + Delete | 清除缓存 |
| Ctrl + PgUp | 切换至前一个标签页 |
| Ctrl + PgDn | 切换至后一个标签页 |
